算法正确性

算法正确性首先要考虑算法规范。

算法规范

算法规范包含以下三点：

  • 算法名称以及算法的参数列表（如：Sort(A)Sort(A)Sort(A)）
  • 算法的输入（如：An array A of n integersAn \ array \ A \ of \ n \ integersAn array A of n integers）
  • 算法的输出（如：A permutation of that array A is sorted(monotonic)A \ permutation \ of \ that \ array \ A \ is \ sorted (monotonic)A permutation of that array A is sorted(monotonic)）

完全正确性

定义：对于任何正确的输入，算法能够终止并且得到正确的输出。
在检查算法正确性的时候，我们通常会将其分为两部分来证明：算法是否终止，算法是否正确输出。对于后面这一部分，我们称之为部分正确性。

部分正确性

定义：对于一个正确的输入，算法一旦终止就会输出正确的结果。

算法正确性的证明

算法正确性的证明通常分为两步：

  • 证明对于一个正确的输入，算法能够终止；
  • 证明算法具备部分正确性。
    算法正确性的证明通常也有两种不同的方法：
  • 对于终止条件的变式；
  • 对于部分正确性的不变式。
